# 6DuckLearn Trademark Guidelines

`6DuckLearn` is a product and brand name owned by its maintainers.

You may:

- use this repository to connect to the hosted 6DuckLearn MCP server
- describe forks as "for 6DuckLearn" or "compatible with 6DuckLearn"
- reference the hosted MCP endpoint in documentation

You may not:

- imply your fork is the official 6DuckLearn service unless it is published by the 6DuckLearn maintainers
- use the 6DuckLearn name or logo in a way that confuses users about who operates a service
- publish credentials, private endpoints, or user data under the 6DuckLearn brand

Official public repositories are currently published under `kit18` unless a dedicated 6DuckLearn organization is created later.
