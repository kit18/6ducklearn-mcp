# 6DuckLearn MCP Tool Use Cases

This catalog describes the public use cases for 6DuckLearn MCP tools. It is a practical guide for users and agents, not a raw schema reference. Availability depends on account authorization, connected client support, and the active approval policy for the workspace.

Use these examples as natural-language prompts. Write-capable actions require account authorization and may be approval-gated.

|---|---|---|---|---|

| `build_with_ai_fetch_data_sources` | Build-with-AI data | Gather selected external or saved data inputs for a user-authorized research task. | "Fetch news and RSS context for this market theme before summarizing it." | Read action that may depend on source availability and account permissions. |
